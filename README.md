# 🔐 Електронний підпис файлу

Простий набір Node.js-скриптів для генерації RSA-ключів, підписання файлів та верифікації цифрового підпису.

---

## 📋 Можливості

- ✅ Генерація пари RSA-ключів (2048 біт) у форматі PEM
- ✅ Підписання файлу за допомогою приватного ключа (SHA-256)
- ✅ Верифікація цифрового підпису за допомогою публічного ключа

---

## 🚀 Встановлення

Проєкт використовує лише вбудовані модулі Node.js — додаткових залежностей немає.

```bash
git clone https://github.com/your-username/rsa-signature-toolkit.git
cd rsa-signature-toolkit
```

> Переконайтеся, що у вас встановлено [Node.js](https://nodejs.org/) версії **14** або вище.

---

## 📁 Структура проєкту

```
rsa-signature-toolkit/
├── generateKeys.js     # Генерація RSA-ключів
├── signFile.js         # Підписання файлу
├── signatureCheck.js   # Верифікація підпису
├── document.txt        # Файл для підписання (ваш документ)
├── public_key.pem      # Публічний ключ (генерується)
├── private_key.pem     # Приватний ключ (генерується)
└── signature.sig       # Підпис файлу (генерується)
```

---

## ⚙️ Інструкція використання
> [!WARNING]
> Потрібно виконувати все саме в такому порядку

### Перший крок: Генерація ключів
Спочатку потрібно запустити generateKeys.js щоб створити файли `public_key.pem` та `private_key.pem`.
```bash
node generateKeys.js
```

> [!WARNING]
> Зберігайте `private_key.pem` у безпечному місці. Ніколи не передавайте його іншим.

---

### Другий крок: Підписання файлу

Покладіть документ, який потрібно підписати, у файл `document.txt`, потім виконайте:

```bash
node signFile.js
```

Створює файл `signature.sig` з цифровим підписом у форматі Base64.

---

### Третій крок: Верифікація підпису
Запустіть signatureCheck.js який перевірить документ та видасть результат на основі того чи змінювався він, чи ні
```bash
node signatureCheck.js
```
## Результат:

| Результат | Повідомлення |
|-----------|-------------|
| ✅ Успіх | `Підпис дійсний. Документ не змінено.` |
| ❌ Помилка | `Підпис недійсний! Документ змінено або ключ неправильний.` |
